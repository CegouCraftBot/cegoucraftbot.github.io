Команды бота

Основные
Инструменты
Игры

.list

Показать онлайн игроков сервера

.stat

Показать статистику игрока. Отображает количество заходов на сервера CegouCraft, привилегии, префиксы, лайки/дизлайки.

Информация о игровом проекте CegouCraft - MOTD, текущий онлайн, ядро, информация о цифровом айпи адресе.

.server

Информация о Discord-сервере

.tabwatcher

Мониторинг игроков через TabWatcher. Предназначен чтобы оповещать в Discord при заходе определенного игрока на сервер.
.tabwatcher help - помощь
.tabwatcher add <nickname> - добавить никнейм
.tabwatcher remove <nickname> - удалить никнейм
.tabwatcher list - показать список никнеймов
.tabwatcher enable - включить функцию
.tabwatcher disable - выключить функцию

.check

Проверка привилегий (бд сайта CegouCraft.ru)

.skins

Скачать скины всех онлайн игроков

.uuid

Узнать UUID игрока

.find

Найти похожие никнеймы (Выживание #1)

.hide

Скрыть информацию о себе
Ваши данные будут скрыты из выдачи в течение 3-20 дней после подачи заявки. Если информация не подтвердится, данные не будут скрыты.

Скрытые никнеймы не будут выданы в командах:
.stat
.skins
.find

Документация API

Информация об API

API CegouCraftBot предоставляет разработчикам доступ ко всем данным и функциям бота. Используйте API для создания собственных инструментов мониторинга, интеграции с вашим сайтом или мобильным приложением.

Хост: http://91.108.244.180:9000/
Версия: 1.1
Релиз: 25.02.2025
Обновление: 28.02.2025

Эндпоинты:

/players/list GET

Получить полный список игроков с Выживания #1 или Анархии

Параметры запроса:
server (string) [обязательный] - Сервер: survival, anarchy, all
Заголовки:
userId (header) [обязательный] - Идентификатор пользователя
Access-Token (header) [обязательный] - Токен доступа

/stats GET

Получить статистику игрока

Параметры запроса:
nickname (string) [обязательный] - Никнейм игрока
Заголовки:
userId (header) [обязательный] - Идентификатор пользователя
Access-Token (header) [обязательный] - Токен доступа
Коды ошибок:
1001: "Отсутствуют данные авторизации"
1002: "Неверные данные авторизации"
1003: "Ошибка обработки данных"
1004: "Внутренняя ошибка сервера"
1005: "Превышен лимит запросов"
1006: "Неизвестный сервер"
1007: "Ошибка загрузки данных viplist/trashlist"
1008: "Никнейм не может быть пустым"
1009: "Игрок скрыт (.hide команда)"

Правила API:

  • 1. Запрещено спамить в API.
  • 2. Запрещено передавать токен API.
  • 3. Запрещено парсить информацию игроков через API (статистику и т.д).

Наказание за нарушение: бан в боте/API (на усмотрение администрации)

Часто задаваемые вопросы

Как добавить бота на мой сервер?

Бот не предназначен для добавления на сторонние Discord-сервера. Он доступен только на официальном Discord-сервере проекта CegouCraftBot: https://dsc.gg/cegou

Как получить API ключ для интеграции?

Для получения API ключа используйте команду .api в личных сообщениях Discord-бота.

Сколько стоит использование бота?

Бот полностью бесплатный. Все функции и возможности доступны всем пользователям без каких-либо ограничений.

Доступен ли исходный код бота?

Исходный код CegouCraftBot не предоставляется - это закрытый проект. Но если вам нужен частичный доступ к возможностям бота, можете воспользоваться нашим API и подключить его к своему проекту.
