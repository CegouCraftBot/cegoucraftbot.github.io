Эндпоинты:

GET /players/list

Получить список игроков с серверов

Параметры запроса:
server (string) [обязательный] - Сервер: survival, anarchy, all
Заголовки:
userId (header) [обязательный] - Идентификатор пользователя
Access-Token (header) [обязательный] - Токен доступа

GET /stats

Получить статистику игрока по никнейму

Параметры запроса:
nickname (string) [обязательный] - Никнейм игрока
Заголовки:
userId (header) [обязательный] - Идентификатор пользователя
Access-Token (header) [обязательный] - Токен доступа
Коды ошибок:
1001: "Отсутствуют данные авторизации"
1002: "Неверные данные авторизации"
1003: "Ошибка обработки данных"
1004: "Внутренняя ошибка сервера"
1005: "Превышен лимит запросов"
1006: "Неизвестный сервер"
1007: "Ошибка загрузки данных viplist/trashlist"
1008: "Никнейм не может быть пустым"
1009: "Игрок скрыт (.hide команда)"
